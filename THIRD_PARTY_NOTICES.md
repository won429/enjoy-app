# Third-Party Software

The random game page includes these open-source components:

- `spin-wheel` 5.0.2 by CrazyTim, MIT License
  - https://github.com/CrazyTim/spin-wheel
  - Full license: `vendor/LICENSE-spin-wheel.txt`
- `@3d-dice/dice-box-threejs` 0.0.12 by Frank Ali, MIT License
  - https://github.com/3d-dice/dice-box-threejs
  - Full license: `vendor/LICENSE-dice-box-threejs.txt`

Both components are used only as presentation engines. Firebase remains the
shared source of truth for participants, choices, predetermined outcomes, and
round state.
